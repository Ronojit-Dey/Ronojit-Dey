<summary><b> Talk to PDF</b> — Local AI document Q&A system</summary>

<br/>

```
Upload any PDF → Embed → Query with natural language → Get grounded answers
No paid APIs. No data leaving your machine.
```

**Stack:** `Python` `Streamlit` `ChromaDB` `Ollama` `HuggingFace`

- Built a full local RAG pipeline: PDF ingestion → chunking → vector embedding → retrieval → LLM response
- Runs entirely offline — designed for live demo without API dependency
- Semantic search via ChromaDB + locally deployed open-source LLM via Ollama

🔗 [View Repository](https://github.com/Ronojit-Dey)
